# 05 Book Edit Page

Goal: rebuild editing as a cinematic media detail workflow.

- Add media detail hero with cover, title, author, rating, duration, save status, and actions.
- Add tabs for Metadata, Files, Chapters, and Suggestions.
- Suggestions use field-level compare and apply.
- Chapters use compact virtualized rows in their own tab.

Acceptance:

- Existing save behavior still works.
- Suggestions no longer overwrite all fields implicitly.
- Chapters no longer dominate the initial edit view.
- Large chapter lists remain usable.
